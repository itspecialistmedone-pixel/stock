LINE Messaging API
ส่งข้อความผ่าน LINE Official Account
User ID ขึ้นต้นด้วย U / Group ID ขึ้นต้นด้วย C / Room ID ขึ้นต้นด้วย R
📋 ขั้นตอนตั้งค่า LINE Messaging API (ทำครั้งเดียว)
1
สร้าง LINE Official Account
ไปที่ developers.line.biz → Log in ด้วย LINE → Create a new provider → Create a Messaging API channel
2
รับ Channel Access Token
เข้า Channel → แท็บ Messaging API → เลื่อนลงหา Channel access token → กด Issue → Copy token แล้ววางในช่องด้านบน
3
หา User ID ของตัวเอง
เข้า LINE Developers Console → เลือก Channel → แท็บ Messaging API → เพิ่มเพื่อน Bot ก่อน (scan QR) → แล้วเช็ค User ID ได้จาก webhook หรือใช้วิธีด้านล่าง
4
วิธีรับ User ID ง่ายๆ
เพิ่มเพื่อน Bot แล้วส่งข้อความใดก็ได้ → เปิด Developers Console → Webhook URL ใส่ https://webhook.site/your-id จาก webhook.site → ดู userId ใน JSON ที่ได้รับ
5
ส่งหา Group / ห้องแชท
เพิ่ม Bot เข้า Group LINE → Bot จะได้รับ groupId (ขึ้นต้น C) → ใช้ webhook.site รับ event แล้วก็อป groupId ใส่ช่อง User/Group ID ด้านบน
⚠️ หมายเหตุสำคัญ: LINE Messaging API ส่งจาก browser ตรงๆ ไม่ได้เพราะ CORS
✅ วิธีแก้: ใช้ Cloudflare Worker เป็น proxy (ฟรี) หรือ Make.com / n8n รับ Webhook แล้วส่งต่อ LINE
👉 กด
ถ้าไม่มี Proxy ระบบจะส่งตรงไป api.line.me (อาจถูกบล็อก CORS บางเบราว์เซอร์)
🗺️ ภาพรวมขั้นตอนทั้งหมด (ทำครั้งเดียว)
Step 1สร้าง LINE Official Account ที่ developers.line.biz
Step 2ไปแท็บ Messaging API → กด Issue รับ Channel Access Token → Copy ใส่ช่องบน
Step 3เพิ่มเพื่อน Bot (scan QR ใน Console) → ส่งข้อความหา Bot ก่อน 1 ข้อความ
Step 4เข้า webhook.site → Copy URL → วางใน Webhook URL ของ LINE Channel → ส่งข้อความหา Bot → ดู userId ใน JSON → Copy ใส่ช่อง User/Group ID
Step 5แก้ CORS: สมัคร make.com → สร้าง Scenario → Webhook รับ → HTTP Request ส่งต่อ LINE → Copy Webhook URL ใส่ช่อง Proxy URL
Step 6กด ทดสอบ ในระบบ — ถ้าขึ้น LINE ✅ แสดงว่าสำเร็จ!
