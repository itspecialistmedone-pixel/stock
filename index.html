✈️
Telegram Bot
ส่งข้อความผ่าน Bot Telegram — คลิกเพื่อตั้งค่า
📡 Flow Telegram ที่ตั้งค่าไว้
📋 Request ใหม่ → Chat 1 (Admin Group)
✅❌ Approve/Reject → Chat ID รายคนของผู้ขอ ก่อน
↳ fallback: Chat 3 → Chat 2 → Chat 1
📋 Admin copy → Chat 1 รับสำเนาเสมอ (ถ้าไม่ใช่ target เดียวกัน)
สร้าง Bot → @BotFather | หา Chat ID → ส่งข้อความหา Bot แล้วใช้ @userinfobot
📊 Chat 1 — Admin Group
รับแจ้ง: Request ใหม่, สรุปสต็อก, หมดอายุ, สำเนา Approve/Reject
📋 Chat 2 — Request ใหม่ (สำรอง)
ถ้าไม่ใส่ จะใช้ Chat 1 แทน
✅❌ Chat 3 — Approve/Reject (Group ผู้ขอ)
ใช้เป็น fallback เมื่อผู้ขอไม่มี Chat ID รายคน
ถ้าไม่ใส่ จะใช้ Chat 2 → Chat 1 แทน
💡 วิธีตั้งค่า Chat ID รายคน (สำหรับ Approve/Reject ส่งตรง)
1. User แต่ละคนส่งข้อความหา Bot ของคุณก่อน 1 ครั้ง
2. เพิ่ม @userinfobot เข้ากลุ่ม หรือส่งข้อความหามัน → จะได้รับ Chat ID
3. Admin กรอก Chat ID ใน Admin → จัดการผู้ใช้ → ✏️ แก้ไข → ช่อง Telegram Chat ID
